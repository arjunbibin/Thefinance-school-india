### 1. Configure Your Backend (Step 4 in Console)
When you see the **"Configure your backend"** screen:
1. **Backend ID**: Enter `finance-school-india-prod`.
2. Click **Next**.

### 2. Associate a Firebase Web App (Step 5 in Console)
1. Select your existing web app: **`nextn`** (or the one matching your project).
2. Click **Finish**.

### 3. Monitor Your Build
Firebase will now start a "Build and Deploy" job. 
1. Go to the **App Hosting** tab in the Firebase Console.
2. You will see a "Pending" or "In Progress" status.
3. Once it turns green, click the **Domain URL** provided to view your live site.

### 4. Making Future Updates
Every time you want to update your live website, simply run these commands in your terminal:
```bash
git add .
git commit -m "Update site content"
git push origin main
```
Firebase will automatically detect the push and redeploy your site within minutes.
